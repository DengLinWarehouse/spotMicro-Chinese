# 机器狗组装与舵机初始值设置

本文档解决一个比标定更早的问题：**舵机在装配前应该先打到什么角度，再装舵盘和连杆。**

如果这一步做错，后续即使 `servo_move_keyboard` 标定数据再准确，也会在装配完成后立刻撞到危险阈值。典型现象是：

- 肩部正常，但大腿和小腿一侧很快打满。
- 机器狗还没进入正常站立姿态，某些关节已经接近 PWM 上下限。
- 标定得到的斜率 `k` 看起来正确，但 `+90` 或 `-90` 对应的 PWM 已经超出安全范围。

根本原因不是标定错了，而是**舵盘安装零位错了**。

---

## 1. 先统一三个概念

### 1.1 舵机机械中心

以本项目使用的 `PDI-HV5523MG` 为例：

- 总行程：`120 deg`
- 机械中心：`60 deg`
- 安全工作脉宽：约 `900-2100 us`

很多人会自然地把所有舵机都先打到 `60 deg` 再装上。对肩关节这通常没问题，但对大腿和小腿这是错误做法。

### 1.2 装配姿态

本文中的装配姿态不是软件里的 `stand` 状态，而是：

- 机器狗四足落地
- 大腿和小腿**几乎共线**
- 整体呈“矗立”状态

这个姿态只用于装配和检查连杆方向，不代表运行时的目标姿态。

### 1.3 运动学零位

软件里的关节 `0 deg` 也不等于“舵机机械中心”。对本项目来说，关节从零位出发的可用角度明显不对称：

| 关节 | 典型站立角度 | 主要工作方向 | 从 0 需要预留的偏移 |
| --- | --- | --- | --- |
| 左侧肩 | 约 0 | 双向较对称 | 中间即可 |
| 右侧肩 | 约 0 | 双向较对称 | 中间即可 |
| 左侧大腿 | `+50 deg` | 主要向正方向摆动 | 约 `+65 deg` |
| 右侧大腿 | `-50 deg` | 主要向负方向摆动 | 约 `-65 deg` |
| 左侧小腿 | `-99 deg` | 主要向负方向摆动 | 约 `-128 deg` |
| 右侧小腿 | `+99 deg` | 主要向正方向摆动 | 约 `+128 deg` |

因此，大腿和小腿在装配时必须**故意偏离机械中心**，把可用行程预留给主要工作方向。

## 2. 初始值设置原则

按关节类别处理，不要 12 个舵机统一打中心。

### 2.1 肩关节

肩关节内旋和外旋的需求相对对称，直接放在机械中心即可：

- 目标角度：`总行程 * 0.50`
- 对 `120 deg` 舵机：`60 deg`

### 2.2 左侧大腿 / 右侧大腿

大腿关节左右是镜像关系：

- 左侧大腿主要需要向正方向摆动，所以装配零位要偏小。
- 右侧大腿主要需要向负方向摆动，所以装配零位要偏大。

推荐预装角度：

- 左侧大腿：`总行程 * 0.25`
- 右侧大腿：`总行程 * 0.75`

对 `120 deg` 舵机：

- 左侧大腿：`30 deg`
- 右侧大腿：`90 deg`

### 2.3 左侧小腿 / 右侧小腿

小腿关节不对称最明显，绝对不能放在 60 度中心位。

- 左侧小腿主要需要向负方向运动，所以装配零位必须偏大。
- 右侧小腿主要需要向正方向运动，所以装配零位必须偏小。

推荐预装角度：

- 左侧小腿：`总行程 * 0.85`
- 右侧小腿：`总行程 * 0.15`

对 `120 deg` 舵机：

- 左侧小腿：`102 deg`
- 右侧小腿：`18 deg`

实际装配时由于花键不能连续调角，可按最接近的齿位安装，通常会落在：

- 左侧小腿：`100-110 deg`
- 右侧小腿：`10-20 deg`

## 4. 为什么不能全部设成 60 度

以左前小腿 `LF_3` 为例：

- 如果你把它装在 `60 deg` 机械中心
- 它向负方向最多只能再走约 `60 deg`
- 但软件站立和步态往往要求它从零位向负方向偏得更多

结果就是：

- 标定时 `-90 deg` 很快落到 PWM 下限
- 站立姿态还没摆出来，舵机就已经接近堵转
- 你看到的是“标定危险”，本质上是“装配零位错误”

左侧小腿要高位装，右侧小腿要低位装；左侧大腿要低位装，右侧大腿要高位装。这个偏置必须在装配前完成。

## 5. 实际装配步骤

### 步骤 1：拆下所有连杆，只保留舵机本体

先不要在整机组装完成后再去硬拧。那样最容易把舵机顶到极限。

### 步骤 2：逐个舵机打到预装角度

可以使用 `SERVO_PAC9685/raspi_pca9685_test` 下的脚本，也可以用 ROS 的 `servo_move_keyboard`。

如果使用独立脚本，建议一次只装一个舵机：

```bash
python3 set_servo.py 1 60
python3 set_servo.py 2 30
python3 set_servo.py 3 102
```

### 步骤 3：在该角度下安装舵盘

目标不是让外壳好看，而是让连杆方向接近目标装配姿态。

判断标准：

- 肩关节装好后，左右摆动余量接近。
- 左大腿装好后，后摆余量明显大于前摆。
- 右大腿装好后，前后方向与左腿镜像。
- 左小腿装好后，下压方向余量更大。
- 右小腿装好后，与左小腿镜像。

### 步骤 4：花键按最近齿位装配

`PDI-HV5523MG` 为 `25T` 花键，每齿约：

```text
360 / 25 = 14.4 deg
```

所以你几乎不可能精确装在 `102 deg` 或 `18 deg`。正确做法是：

- 先选最接近的齿位
- 再通过后续标定和 `center_angle_deg` 微调

不要试图靠软件补偿一个明显错了 2-3 齿的舵盘位置。

### 步骤 5：装成“矗立姿态”后再进入标定

装配完成后，整机应满足：

- 四足支撑
- 大腿和小腿几乎共线
- 不进入软件 `stand` 状态也不会有明显关节顶死

只有满足这个前提，后面的 `servo_move_keyboard` 标定才有意义。

## 6. 一眼判断装错的方法

装完后，如果出现以下情况，优先怀疑舵盘零位，而不是先改 YAML：

- 左大腿一动就朝极限方向跑
- 左小腿在 `-90 deg` 附近已经接近 PWM 下限
- 右侧同类关节和左侧不呈镜像
- 机器人还没进入正常站立姿态，小腿或大腿已经明显发热、抖动、嗡鸣

对应处理方式：

- 肩关节异常：重装到接近中心位
- 左大腿余量不足：舵盘再往小角度方向退 `1-2` 齿
- 右大腿余量不足：舵盘再往大角度方向进 `1-2` 齿
- 左小腿余量不足：舵盘再往大角度方向进 `1-2` 齿
- 右小腿余量不足：舵盘再往小角度方向退 `1-2` 齿

---

## 7. 推荐工作流

正确顺序如下：

1. 确认舵机型号与总行程
2. 计算预装角度
3. 单舵机打角度
4. 装舵盘和连杆
5. 形成“矗立姿态”
6. 再做 `servo_move_keyboard` 标定
7. 最后写入 `spot_micro_motion_cmd.yaml`

不要反过来做。**如果装配零位错了，标定会被迫在危险区工作。**

---

## 8. 本项目当前建议

对你现在这套 `PDI-HV5523MG 120 deg` 方案，建议直接从下面这组值开始：

```text
肩：60
左大腿：30
右大腿：90
左小腿：102
右小腿：18
```

四条腿按左右镜像复用即可：

- 左前 / 左后：`60, 30, 102`
- 右前 / 右后：`60, 90, 18`

如果上机后仍有轻微偏差，优先通过“调一齿”修正，再进入软件标定。不要一上来就靠扩大 PWM 范围硬顶。
