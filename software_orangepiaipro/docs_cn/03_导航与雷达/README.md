# 03_导航与雷达

本目录收纳雷达接入、建图、地图保存、AMCL 定位、导航方案与相关排障文档。

- `雷达使用手册.md`：雷达正式使用入口
- `无RViz纯命令行导航测试手册.md`：无 GUI 场景下的测试手册
- `SpotMicro_雷达避障漫游自动建图方案.md`：仅依赖雷达的自动探索建图方案
- `SpotMicro_两条命令统一启动方案.md`：自动探索建图 / 自动巡视的一键启动设计
- `自动探索与自动巡视运行手册.md`：自动探索建图 / 自动巡视的统一运行入口
- `自动探索转向步态逐渐混乱问题记录与修复方案.md`：自动探索遇障转向时步态逐渐混乱、压腿、侧翻问题的记录与修复说明
- `自动探索遇障频繁切换问题与模糊控制方案可行性分析.md`：记录自动探索中遇障后频繁切换前进/转向的问题，并分析引入滞回、去抖和平滑状态机/模糊控制思想的可行性
- `../05_香橙派运行负载分析/香橙派CPU绑核_高负载_建图卡顿分析报告.md`：CPU 绑核、第四核未参与、高负载与卡顿问题分析
- `../05_香橙派运行负载分析/解除0-2绑核并重新启动建图操作手册.md`：解除当前 shell 的 0-2 绑核并重新验证建图链路
- `../05_香橙派运行负载分析/方案A_脚本级永久解除0-2绑核优化设计.md`：将解绑核固化到机器人启动脚本中的长期方案
- `SpotMicro_雷达建图与AMCL定位阶段总结.md`：阶段总结
- `RPLidar_Hector_实操问题复盘.md`：现场问题复盘
- `AMCL_move_base_源码补齐方案.md`：源码补齐方案
- `navigation_stack_runbook.md`：运行手册
- `SpotMicro 导航方案设计.md`：导航总体设计
- `tf_chain_spotmicro_navigation.md`：导航 TF 说明

建议阅读顺序：

1. `雷达使用手册.md`
2. `无RViz纯命令行导航测试手册.md`
3. `navigation_stack_runbook.md`
4. 需要深入时再看设计文档与复盘文档
